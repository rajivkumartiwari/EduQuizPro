Flashcard Review Decks

Double-sided concept review cards with interactive test games to test vocabulary, hardware specs, and systems.

Deck 1: Input Hardware

Review key input components like scanners, keyboards, and mice.

Deck 2: Output Hardware

Inspect printers, monitors, speakers, and projectors.

Deck 3: Storage Disks

Review flash USBs, hard drives, and SSD memory chips.

Deck 4: Special Scanners

Examine OCR scanners, MICR check readers, and barcode sensors.

Deck 5: Mobile Devices

Study tablets, touchscreens, laptops, and smartphones.

Deck 6: Web Code

Review APIs, CSS rules, HTML tags, and JavaScript files.

Deck 7: System Platforms

Learn macOS systems, Linux kernels, Windows, and Android OS.

Deck 8: Net Devices

Review modems, switches, LAN routers, and NIC cards.

Deck 9: Storage Scales

Study bits, characters, bytes, and megabytes structures.

Deck 10: Threat Signs

Examine popups, lagging screens, phishing emails, and open links.
